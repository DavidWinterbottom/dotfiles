dotfiles.git
============
```
cd $HOME
git clone https://github.com/DavidWinterbottom/dotfiles.git .dotfiles
ln -sb .dotfiles/.bash_profile .
ln -sb .dotfiles/.bashrc .
ln -sb .dotfiles/.bashrc_custom .
ln -sb .dotfiles/.screenrc .
ln -sb .dotfiles/.gitconfig .
```
